# 🐚 TP 5 – Scripting Shell (Bash)

Bienvenue dans le cinquième travail pratique du cours **Administration Système sous Linux**.

Ce TP vous permettra d'apprendre à écrire vos premiers scripts Bash afin d'automatiser des tâches sous Linux.

---

## 🎯 Objectifs

À la fin de ce TP, vous serez capable de :

- Créer un script Shell.
- Exécuter un script Bash.
- Utiliser des variables.
- Lire des données au clavier.
- Utiliser des paramètres.
- Écrire des conditions simples.
- Utiliser des boucles.
- Créer une fonction simple.

---

## 📋 Prérequis

Avant de commencer :

- 💻 Ubuntu ou Kali Linux
- 👤 Compte utilisateur
- 🖥️ Terminal Linux
