# 💾 TP 4 – Gestion du système de fichiers (suite)

Bienvenue dans le quatrième travail pratique du cours **Administration Système sous Linux**.

Ce TP vous permettra de découvrir l'administration des systèmes de fichiers, le montage des partitions et la gestion de l'espace disque.

---

## 🎯 Objectifs

À la fin de ce TP, vous serez capable de :

- Monter et démonter un système de fichiers.
- Surveiller l'espace disque.
- Gérer les partitions Linux.
- Utiliser les systèmes de fichiers.
- Comprendre les bases du LVM.

---

## 📋 Prérequis

Avant de commencer :

- 💻 Ubuntu ou Kali Linux
- 🖥️ VirtualBox
- 👤 Compte sudo
