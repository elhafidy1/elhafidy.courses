# 🐧 TP 1 – Installation et Configuration de VirtualBox et Ubuntu/Kali

Bienvenue dans le premier travail pratique du cours **Administration Système sous Linux**.

Ce TP vous permettra de préparer votre environnement de travail en installant Oracle VirtualBox puis une machine virtuelle Ubuntu ou Kali Linux.

---

## 🎯 Objectifs

À la fin de ce TP, vous serez capable de :

- Télécharger Oracle VirtualBox.
- Installer Oracle VirtualBox sous Windows.
- Télécharger l'image ISO d'Ubuntu ou Kali Linux.
- Créer une machine virtuelle.
- Configurer la mémoire RAM et le disque virtuel.
- Installer Ubuntu ou Kali Linux.
- Vérifier le bon fonctionnement de la machine virtuelle.

---

## 📋 Prérequis

Avant de commencer, assurez-vous de disposer de :

- 💻 Un ordinateur sous **Windows 10** ou **Windows 11**
- 🧠 Au moins **8 Go de mémoire RAM** (recommandé)
- 💾 Au moins **25 Go d'espace disque libre**
- 🌐 Une connexion Internet

## 💡 Conseils

Pendant ce TP, prenez le temps de :

- Vérifier que la virtualisation est activée dans le BIOS.
- Attribuer suffisamment de RAM à votre machine virtuelle.
- Installer les **Guest Additions** de VirtualBox.
- Faire un **Snapshot** après l'installation afin de pouvoir restaurer facilement votre machine en cas de problème.
